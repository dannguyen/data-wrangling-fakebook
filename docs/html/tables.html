Tables¶

Simple tables¶

  • First column cells cannot be multiple lines
  • Must contain more than 1 row
name age
Alice 42
Bob 99 
=======  =====
name     age
=======  =====
Alice    42
Bob      99
=======  =====

Grid Table¶

+-------+-----+
| name  | age |
+=======+=====+
| Alice | 42  |
+-------+-----+
| Bob   | 99  |
+-------+-----+

Result:

name age
Alice 42
Bob 99
